Espresso Machine Error Code Lookup

Error codes are the machine telling you what's wrong. Use this lookup to decode the message — DeLonghi, Saeco, Jura, Breville, Gaggia, La Pavoni, Rocket, Lelit.

  • DeLonghi Magnifica — Error Code Table

    DeLonghi uses four-digit numeric codes on Magnifica, Eletta, Dinamica, and PrimaDonna super-automatics. Full context for these codes is in the DeLonghi Magnifica troubleshooting guide.

    CodeMeaningFirst Fix
    1101General Alarm (catch-all)Power cycle, remove/reinsert brew unit, inspect sensor
    1026General Alarm / dry groundsClean brew unit, replace infuser O-rings
    1454Grinder stuckEmpty bean hopper, vacuum grinder chamber, coarsest setting
    1512General alarm (Magnifica S)Brew unit / position sensor
    2257No water (Magnifica S)Prime pump via hot water spout, descale
    2262No water (Magnifica XS)Same as 2257
    3963Leak alarmInspect all seals, tray, brew unit
    3984Alarm (Magnifica EVO)Sensor or brew unit fault
    5Coffee too fine / grind errorDial grind coarser by 1 step
    Empty CircuitFlowmeter sees no flow (PrimaDonna)Prime via hot-water spout until flow steady

  • Saeco — Error Code Table

    Saeco uses alphanumeric codes (E01, E03, etc.) plus numeric "brew group" codes like 268. See the Saeco troubleshooting hub and the dedicated Saeco Error 268 guide.

    CodeMeaningFirst Fix
    268Brew group not homingRemove brew group, clean, reinsert fully
    E01Grinder jamEmpty hopper, vacuum grinder, coarsest setting
    E02Brew group jamRemove brew group, rinse, re-install
    E03Brew group position faultBrew group sensor or motor; often needs service
    E04Water circuit overflowEmpty drip tray, check 3-way valve
    E05Water circuit empty / no flowPrime circuit, descale, check pump
    E13Flowmeter / no waterDescale, inspect flowmeter, re-prime
    E14Thermoblock overheatLet cool, check for scale blockage, descale
    E15Boiler sensor faultService required — NTC sensor replacement

  • Jura — Error Code Table

    Jura uses simple single-digit codes (1–16) or words. See the Jura troubleshooting guide for model-specific walkthroughs.

    CodeMeaningFirst Fix
    1Empty water tankRefill tank, reseat firmly
    2Fill beansRefill hopper, wipe bean chute
    3Empty grounds containerEmpty dreg bin, dry, reinsert
    6Milk system needs cleaningRun milk clean cycle with Jura tablets
    8Brew group fault / generalPower cycle; if persists, service required
    16Water system empty / priming neededOpen spigot, prime with hot water, re-run
    CClean neededRun cleaning tablet cycle
    DescaleScale threshold reachedRun descale cycle with Jura descaling tablets

  • Breville — Error Indicators (Visual)

    Breville semi-automatic machines (Barista Express, Bambino, Dual Boiler, Oracle) don't use numeric error codes — they use button flash patterns. See the Breville Barista Express guide for photos and button-pattern meanings.

    PatternMeaningFirst Fix
    All buttons flashing rapidlyFirmware or power faultUnplug 60s, plug back in
    1-cup + 2-cup flashing alternatingOverheat / thermal cutoutLet cool 30 min, descale if overdue
    Steam + hot-water flashingBoiler heating faultCheck heating element, book service
    Power button dim, no responsePower supply / control boardTry different outlet; likely service needed
    Grind button flashingGrinder hopper empty or jammedRefill beans, clear grinder chamber
    Filter icon litWater filter replacement dueInstall fresh Breville filter cartridge

  • Gaggia Accademia / Anima — Error Codes

    Gaggia super-automatics share architecture with Saeco (Philips owns both), so error codes overlap substantially.

    CodeMeaningFirst Fix
    Empty drip trayFloat sensor triggeredEmpty drip tray, dry sensor area
    Brew group errorJam or position fault (= Saeco 268)Remove brew group, clean, reinsert
    DescaleScale thresholdRun descale with Gaggia descaler
    Grinder blockedForeign object or oil jamVacuum chamber, coarsest setting, test
    Temperature errorNTC sensor or heater faultService required

  • La Pavoni — Lever Machine Indicators

    La Pavoni lever machines (Europiccola, Professional) are mostly indicator lights rather than error codes.

    IndicatorMeaningAction
    Ready light offHeating element fault or pressurestatTest heater continuity; replace pressurestat if cycling fails
    Pressure alarm / relief valve hissingOver-pressure ventingNormal at startup; sustained = pressurestat stuck
    Ready light stays on, no heatRelay or pressurestatService required

  • Rocket / E61 Group Machines

    Rocket Appartamento, Mozzafiato, R58/R60 use a classic E61 grouphead with pressurestat-controlled heating. Error modes are mostly thermodynamic rather than electronic.

    SymptomCauseFix
    No heat, boiler coldPressurestat stuck open or blown thermal fuseTest pressurestat, replace thermal fuse
    Boiler over-pressure (constant safety valve hiss)Pressurestat stuck closedReplace pressurestat (Sirai or Mater)
    Group leaks continuouslyE61 group rebuild neededGasket + mushroom + cam spring rebuild kit
    No flow from groupThermosiphon airlock or solenoid failurePrime via hot-water tap, inspect 3-way solenoid

  • Lelit — PID Error Codes

    Lelit Bianca, Mara X, Elizabeth, and Anita use PID controllers that display specific error codes on the digital panel. See the Lelit troubleshooting hub.

    CodeMeaningFirst Fix
    Er1 / Err1Boiler NTC sensor faultCheck NTC wiring, replace sensor
    Er2 / Err2Group NTC fault (Mara X / Bianca)Check group sensor connection
    Er3 / Err3Overheat / safety thermostatLet cool, check heater relay stuck-on
    Flow control errorFlowmeter not detecting flow (Bianca)Descale, prime, inspect flowmeter wiring
    tStSelf-test mode (not an error)Exit by power cycling

  • La Marzocco

    La Marzocco Linea Mini, GS3, and commercial machines log errors via the Gicar control box. See the La Marzocco troubleshooting hub.

    SymptomCauseAction
    Boiler not filling (water level LED flashing)Probe fouled or autofill solenoid failedClean level probe, test solenoid
    Rotary pump noisyAir in line or worn vanesBleed line, replace pump if sustained
    Temp unstablePID or SSR faultService required — diagnostic via Gicar box
    Steam won't buildElement, relay, or pressurestatTest each in sequence
